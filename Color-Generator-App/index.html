How to Use

  1. Generating Colors:
    • Open the app and click on the "Generate" button.
    • 5 new color codes will instantly appear on the screen.
  2. Copying Color Codes:
    • Click on any color code displayed on the screen.
    • The selected color code will be copied to the clipboard, and a message will appear indicating that the color code has been successfully copied.
